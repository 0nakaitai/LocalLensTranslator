# LocalLensTranslator

LM Studio（ローカルLLM）を使って、画面上の任意の範囲をOCRで読み取りリアルタイム翻訳するWindowsデスクトップツールです。

---

## 必要環境

- Windows 10 / 11
- Python 3.10 以上
- LM Studio（ローカルサーバーとして起動しておく必要があります）

---

## インストール

```bash
pip install mss Pillow winsdk
```

ボーダーレス化機能を使う場合は追加で：

```bash
pip install pywin32
```

グローバルショートカットを使う場合は追加で：

```bash
pip install keyboard
```

---

## 起動方法

```bash
python main.py
```

---

## 基本的な使い方

1. LM Studio を起動し、ローカルサーバーをONにする
2. LocalLensTranslator を起動する
3. **⚙設定** からLM StudioのエンドポイントURLとモデルを設定して保存する
4. **＋ 枠を追加** ボタンを押し、翻訳したい画面範囲をドラッグで選択する
5. 自動的にOCR→翻訳が実行され、選択した範囲に翻訳結果が重なって表示される

---

## 画面説明

### メインウィンドウ

| 要素 | 説明 |
|------|------|
| ● 翻訳 ON / ○ 翻訳 OFF | 全枠の翻訳表示のON/OFFインジケーター |
| ドット最適化 | ドット絵フォント向けのOCR前処理を有効にする |
| 移動再翻訳 | 枠を移動したとき新しい位置で自動再翻訳する |
| ＋ 枠を追加 | 翻訳範囲を新規選択する |
| ▶ 全枠を翻訳 | すべての枠を再翻訳する |
| ⏸ ON/OFF | 翻訳表示をまとめてON/OFFする |
| ✕ 全枠クリア | すべての翻訳枠を削除する |
| ⚙ | 設定ダイアログを開く |
| 翻訳枠一覧 | 現在の翻訳枠の座標と状態の一覧 |
| OCR補正辞書 | OCR誤認識を補正するルールを記述する欄 |
| 対象をボーダーレス化 | アクティブなゲームウィンドウをボーダーレス全画面にする |
| 翻訳元トグル | 翻訳元の言語を切り替える（日→英 / 英語 / 中国語 / 韓国語） |

### 翻訳枠（オーバーレイ）の操作

| 操作 | 動作 |
|------|------|
| 右クリック | メニューを表示（再翻訳 / 日→英翻訳してコピー / ☒移動再翻訳 / 削除 / ON-OFF） |
| 右クリック＋ドラッグ | 枠を移動する（移動再翻訳がONの場合は新位置で再翻訳） |

## 翻訳元言語

| トグル | 動作 |
|--------|------|
| 日→英 | 日本語をOCRで読み取り英語に翻訳。結果をクリップボードにコピーする |
| 英語 | 英語をOCRで読み取り日本語に翻訳する |
| 中国語 | 中国語をOCRで読み取り日本語に翻訳する |
| 韓国語 | 韓国語をOCRで読み取り日本語に翻訳する |

## OCR補正辞書

OCRが誤認識しやすい単語を手動で補正できます。

**書き方：** `誤認識語=正しい語`（1行1エントリ、`#` で始まる行はコメント）

```
# よくある誤認識の補正例
Clamage=Damage
Mp=HP
0rk=Ork
```

- 💾 ボタン：辞書をテキストファイルとして保存する
- 📂 ボタン：保存した辞書ファイルを読み込む

ゲームごとに辞書ファイルを保存しておくと使い回しができて便利です。

## ⚙ 設定

| 項目 | 説明 |
|------|------|
| エンドポイントURL | LM Studio のAPIアドレス（デフォルト: `http://127.0.0.1:1234/v1`） |
| モデル名 | 使用するモデル（「取得」ボタンでLM Studioから一覧取得できる） |
| 翻訳プロンプト | system プロンプトの末尾に追加する補足指示。固有名詞の扱いや口調を指定できる |
| 翻訳プロンプトを使用する | チェックOFFでプロンプト欄の内容を無視する |
| フォント / フォントサイズ | 翻訳結果の表示フォント |
| ショートカットキー | グローバルホットキーの割り当て（✕ボタンで解除） |
| 背景の透明度 | オーバーレイの透明度（10%〜100%） |
| 色の取得方法 | 画像から自動抽出 / 手動で背景色・文字色を指定 |

### 翻訳プロンプトの書き方

プロンプト欄には翻訳スタイルの指示を英語で書くのが確実です。

```
Do not translate proper nouns such as "Survivor". Keep them as is.
Use polite Japanese (丁寧語) for the translation.
```

## ボーダーレス化機能

ボーダーレスウィンドウに対応していないゲームを強制的にボーダーレス全画面にする機能です。

1. **「対象をボーダーレス化」** ボタンを押す
2. 3秒以内にゲームのウィンドウをクリックしてアクティブにする
3. ゲームウィンドウがボーダーレス全画面になる
4. 元に戻す場合は再度ボタンを押す（1280×720のウィンドウに戻る）

> `pip install pywin32` が必要です。

---

## ショートカットキー（デフォルト）

| キー | 動作 |
|------|------|
| `Ctrl+Shift+T` | 翻訳 ON/OFF |
| `Ctrl+Shift+A` | 翻訳枠を追加 |
| `Ctrl+Shift+R` | 全枠を再翻訳 |
| `Ctrl+Shift+C` | 全枠をクリア |

設定ダイアログから変更できます。

---

## 設定の保存先

```
%APPDATA%\LocalLensTranslator\settings.json
```

---

## 注意事項

- 排他フルスクリーン（DirectXフルスクリーン）のゲームではオーバーレイが背面に隠れる場合があります。ゲーム側を**ボーダーレスウィンドウモード**に設定してください。
- Windows OCR は `pip install winsdk` で使用可能になります。未インストールの場合は Tesseract にフォールバックします。
- 翻訳元言語の言語パックはWindowsの⚙設定→時刻と言語→言語と地域から個別に追加ダウンロードする必要があります。
- LM Studio のサーバーが起動していない場合は翻訳エラーになります。
